Thanks for participating in this HIT!

An AI System has performed a task called natural language inference. The system has predicted an answer describing the relationship between two sentences, and then given two different explanations to justify it.

For each pair of sentences and relationship, you will select the best explanation out of the two.


Each HIT contains 4 parts:

PremiseSentence describing a real-world situation, always assumed to be true.
Hypothesis An assumption or inference that we might make about the situation having read the premise.
RelationshipWhether the Hypothesis assumption or inference is valid given the Premise.
ExplanationsTwo explanations for the Relationship.

There are 3 possible Relationships:

trueThe hypothesis is assumed or implied from the premise.
falseThe hypothesis is not assumed or implied from the premise. It contradicts the premise.
neitherThe hypothesis is possible from the premise, but not directly implied or contradicted. More context is needed.

A couple of notes:

  • You may disagree with the Relationship, but you should pretend it is correct when selecting the best Explanation.
  • You can ignore minor grammar and spelling mistakes. Some sentences will be lowercased incorrectly; please ignore this.
  • If you find the Explanations similarly good (or bad), select the one you think is the best and then select "I voted, but this one is close because the explanations are similar."
  • If you find the Explanations equally good (or bad) and cannot choose, select "Both equally good." or "Both equally bad."
